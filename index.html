Récapitulatif du cour HTML5

Qu’est-ce que html ?
HTML est le langage de balisage standard pour la création de pages Web.

  • HTML signifie Hyper Text Markup Language
  • HTML décrit la structure des pages Web à l’aide du balisage
  • Les éléments HTML sont les blocs de construction des pages HTML
  • Les éléments HTML sont représentés par des balises
  • Les balises HTML étiquettent des éléments de contenu tels que « en-tête », « paragraphe », « tableau », etc...
  • Les navigateurs n’affichent pas les balises HTML, mais les utilisent pour afficher le contenu de la page

Structure d'une page HTML

  1. Déclaration du document comme HTML5
  2. Entête : head
  3. Corps : body

Récapitulatif du cour CSS

Qu’est-ce que le CSS ?
CSS est une feuille de style en cascade,qui forme le langage informatique qui décrit la présentation des documents HTML et XML.

  • CSS signifie Cascading Style Sheets
  • CSS est un langage coté client
  • CSS est un langage déscriptif
  • CSS permet de contrôler de manière précise comment un site web est présenté

Structure d'une page CSS

  1. Déclaration du document avec l'extension .css
  2. Une règle CSS se décompose en un sélecteur et une ou plusieurs déclarations appelées bloc de déclaration se trouvant entre les accolades
